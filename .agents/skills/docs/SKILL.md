# Docs

Maintain the root `README.md`.

## Decision rule

1. Inspect files changed in the session.
2. If all changes are under `.agents/`, do not edit `README.md`.
3. If any change is outside `.agents/`, update `README.md` to reflect user-visible impact.

## Guidelines

- Keep content concise and accurate.
- Focus on setup, usage, and behavior changes.
- Avoid documenting internal-only churn.
